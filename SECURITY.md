# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅ Active support |
| < 1.0   | ❌ No longer supported |

## Reporting a Vulnerability

Report security vulnerabilities to: security@performancelab.software

## Security Measures

- No hardcoded credentials
- Input validation on all endpoints
- Authentication for sensitive operations
