# SPRINT — Matemática Adaptativa

> **⚡ Core Engine privado** — O motor de algoritmos adaptativos, geração de exercícios por IA e pipeline de dados está em repositório privado.  
> Este repositório público contém a interface Android, API de superfície e exemplos.

---

SPRINT é um aplicativo de matemática adaptativa que personaliza exercícios conforme o desempenho do aluno, usando IA para gerar desafios no ponto certo de dificuldade.

## Funcionalidades (Superfície Pública)

- 🧮 Exercícios de matemática adaptativos
- 📱 App Android nativo (Kotlin)
- 📊 Dashboard de progresso
- 🎯 Metas e recompensas
- 🔄 Sincronização offline-first

## ⚙️ Core Privado (não incluso aqui)

| Módulo | Descrição | Repo |
|--------|-----------|:----:|
| 🧠 Algoritmo Adaptativo | Motor de dificuldade dinâmica, mastery scoring | 🔒 Privado |
| 🤖 Geração por IA | Criação de exercícios personalizados via LLM | 🔒 Privado |
| 📈 Analytics | Predição de desempenho, recomendação de conteúdo | 🔒 Privado |
| 🔬 Engine de Correção | Análise de erros, feedback adaptativo | 🔒 Privado |

## Stack Pública

`Kotlin` `Python (FastAPI)` `PostgreSQL` `Redis` `Android SDK`
